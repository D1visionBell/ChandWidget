# چند؟! — ویجت قیمت بازار ایران

ویجت اندروید برای نمایش لحظه‌ای قیمت‌های بازار ایران روی صفحه اصلی گوشی — الهام گرفته از اپ Chand برای iOS.

---

## ویجت‌ها

سه نوع ویجت برای سلیقه‌های مختلف:

**ویجت 2×2 تک‌نماد**
یک قیمت به شکل درشت و خوانا. مناسب وقتی می‌خوای یک نماد مهم رو همیشه جلوی چشمت باشه.

**ویجت 2×2 سه‌ردیف**
سه قیمت فشرده در یک ویجت کوچک. ایکون و نماد اختصاری در کنار هم، تغییر قیمت و عدد اصلی سمت راست.

**ویجت 4×2 سه‌ردیف**
سه قیمت با متن بزرگ‌تر و فضای بیشتر. نام کامل نماد، تغییر قیمت و عدد اصلی.

## امکانات

- **۲۰ نماد** از سه دسته ارز، طلا و رمزارز
- **آپدیت خودکار** هر ۳۰ دقیقه
- **آپدیت دستی** با لمس روی ویجت
- **دو زبان** فارسی و انگلیسی — قابل تغییر از صفحه تنظیمات
- **پشتیبانی از دارک مود** — ویجت رنگ پس‌زمینه رو از سیستم می‌گیره
- **تغییر قیمت** با فلش بالا/پایین و رنگ قرمز/سبز
- **هر ویجت مستقل** — می‌تونی چند ویجت با نمادهای مختلف روی صفحه داشته باشی
- **نمایش قیمت‌ها** به تومان برای ارزها و طلا، به دلار برای رمزارزها
